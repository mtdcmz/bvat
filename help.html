API 使用说明

我们提供了一个简单的 API 接口，用于获取视频信息。您可以通过 BV 号来查询对应的视频标题和 MD5 值。

接口地址

API 接口地址为：https://bvat.mtdcmz.workers.dev/

请求参数

使用 GET 方法，通过 URL 参数传递 BV 号：

https://bvat.mtdcmz.workers.dev/?bvid=BV1bs411U726

返回格式

API 返回 JSON 格式的数据，包含以下字段：

  • bvid: 视频的 BV 号
  • title: 视频标题
  • md5: 视频文件的 MD5 值

示例

请求：

https://bvat.mtdcmz.workers.dev/?bvid=BV1bs411U726

返回：

{
    "title": "【暴走敖尼玛】01香菜逼的厂长自爆性取向",
    "bvid": "BV1bs411U726",
    "md5": "b1395c6d2ee2694bd5d1c9d11c72132d"
}

错误处理

如果未找到匹配的视频，API 将返回一个错误消息：

{
    "error": "未找到匹配的视频"
}

使用限制

目前该 API 仅供测试使用，暂无使用限制。在未来可能会增加访问频率限制，请合理使用。

注意事项

1. 该 API 是基于 Cloudflare Workers 实现的，因此请求处理速度依赖网络状况。

2. 数据更新可能不会非常及时，请以实际返回结果为准。

3. 如遇到任何问题，请及时与我们联系。
